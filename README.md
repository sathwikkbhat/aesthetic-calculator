# Aesthetic Calculator

A beautifully designed calculator with a retro aesthetic, built with HTML, CSS, and JavaScript.

## Features

- Clean and attractive retro design
- Full calculator functionality (addition, subtraction, multiplication, division)
- Error handling for invalid operations
- Responsive design

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to Use

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Start using the calculator

## Files

- `index.html` - The main HTML structure
- `style.css` - Styling and design
- `calculator.js` - Calculator functionality

## Preview

The calculator has a unique retro aesthetic with gradient buttons and a vintage look.
